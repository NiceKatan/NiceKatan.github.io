「 XOR Queries of a Subarray 」

位运算 前缀和 Medium 每日一题

  • 题目：有一个正整数数组 arrarrarr，现给你一个对应的查询数组 queriesqueriesqueries，其中 queries[i]=[Li,Ri]queries[i] = [L_i, R_i]queries[i]=[Li​,Ri​]。对于每个查询 iii，请你计算从 LiL_iLi​ 到 RiR_iRi​ 的 XORXORXOR 值（即 arr[Li]⊕arr[Li+1]⊕...⊕arr[Ri]arr[L_i] \oplus arr[L_{i+1}] \oplus ... \oplus arr[R_i]arr[Li​]⊕arr[Li+1​]⊕...⊕arr[Ri​]）作为本次查询的结果。并返回一个包含给定查询 queriesqueriesqueries 所有结果的数组。
  • 示例：

输入：arrarrarr = [1, 3, 4, 8], queriesqueriesqueries = [ [0, 1], [1, 2], [0, 3], [3, 3] ]
输出：[2, 7, 14 ,8]
解释：查询的 XOR 值为：
[0, 1] = 1 xor 3 = 2
[1, 2] = 3 xor 4 = 7
[0, 3] = 1 xor 3 xor 4 xor 8 = 14
[3, 3] = 8

前缀和异或数组

定义 n+1n+1n+1 长度的前缀和数组 preprepre，其中 nnn 为 arrarrarr 数组大小，preprepre 中记录原数组中每一位元素下标范围内所有元素异或结果，之后根据 queriesqueriesqueries 数组中给定范围进行查询

  • preprepre 中元素计算公式：pre[i+1]=pre[i]⊕arr[i]pre[i+1] = pre[i]\oplus arr[i]pre[i+1]=pre[i]⊕arr[i]，pre[i+1]pre[i+1]pre[i+1] 中保存的是 arrarrarr 数组中 [0:i][0:i][0:i] 范围内所有元素的异或结果
