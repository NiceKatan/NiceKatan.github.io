「E」733.图像渲染

发表于2021-07-08更新于2021-07-08字数统计546阅读时长4分

「 Flood Fill 」

深度优先搜索 Easy

  • 题目：有一幅以二维整数数组表示的图画，每一个整数表示该图画的像素值大小，数值在 0 到 65535 之间。给你一个坐标 (sr,scsr, scsr,sc) 表示图像渲染开始的像素值（行 ，列）和一个新的颜色值 newColornewColornewColor，让你重新上色这幅图像。

    为了完成上色工作，从初始坐标开始，记录初始坐标的上下左右四个方向上像素值与初始坐标相同的相连像素点，接着再记录这四个方向上符合条件的像素点与他们对应四个方向上像素值与初始坐标相同的相连像素点，重复该过程。将所有有记录的像素点的颜色值改为新的颜色值。最后返回经过上色渲染后的图像。

  • 示例：

输入：imageimageimage = [ [1, 1, 1], [1, 1, 0], [1, 0, 1] ], srsrsr = 1, scscsc = 1, newColornewColornewColor = 2
输出：[ [2, 2, 2], [2, 2, 0], [2, 0, 1] ]
解释：在图像的正中间，(坐标(sr,sc)=(1,1))，在路径上所有符合条件的像素点的颜色都被更改成2。
注意：右下角的像素没有更改为2，因为它不是在上下左右四个方向上与初始点相连的像素点。

深度优先搜索

从给定坐标开始，DFS上下左右四个方向，判断是否与初始坐标原像素值相同，相同则渲染为新像素值。搜索终止条件为：越界、不等于原像素值、等于新像素值

代码

  • 时间复杂度：O(n×m)O(n\times m)O(n×m)
  • 空间复杂度：O(n×m)O(n\times m)O(n×m)
