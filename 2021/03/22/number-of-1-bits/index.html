「E」191.位1的个数

「 Number Of 1 Bits 」

位运算 Easy 每日一题

  • 编写一个函数，输入是一个无符号整数（以二进制串的形式），返回其二进制表达式中数字位数为 ‘1’ 的个数（也被称为汉明重量）。

  • 示例：

输入：00000000000000000000000000001011
输出：3

通过多次n&(n-1)操作消除二进制串中最右边一位1，直到nnn为0，循环次数就是1的个数。

例如：n=10110，n-1=10101，相&之后n=10100

代码

  • 时间复杂度：O(logn)O(logn)O(logn)
  • 空间复杂度：O(1)O(1)O(1)
