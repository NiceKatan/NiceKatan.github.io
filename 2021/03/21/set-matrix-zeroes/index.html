「 Set Matrix Zeroes 」

数组 Medium 每日一题

  • 题目：给定一个m*n的矩阵，如果一个元素为0，则将其所在行和列的所有元素都设为0。请使用原地算法。

  • 示例：

输入：matrix = [[1,1,1],[1,0,1],[1,1,1]]
输出：[[1,0,1],[0,0,0],[1,0,1]]

方法一：复制数组-O(MN)O(MN)O(MN)

  • 复制数组为temptemptemp并遍历temptemptemp，遇到0将martixmartixmartix对应行列置零，修改原始数组数值。

代码

  • 时间复杂度：O(MN∗(M+N))O(MN*(M+N))O(MN∗(M+N))
  • 空间复杂度：O(MN)O(MN)O(MN)

方法二：数组记录-O(M+N)O(M+N)O(M+N)

  • 分别开辟与行列大小相同的两个数组，遍历数组，记录该行、该列是否需要置零。
  • 再遍历数组，如果容器中对应行、列被标记，数值置零。

代码

  • 时间复杂度：O(MN)O(MN)O(MN)
  • 空间复杂度：O(M+N)O(M+N)O(M+N)

方法三：首行首列记录-O(1)O(1)O(1)

  • 用两个常量标记首行、首列是否需要清零，并用首行、首列记录行列是否清零(原地操作首行首列存在被污染的风险)

具体实现

  • 遍历首行、首列，存在0则置rowrowrow、columncolumncolumn为truetruetrue。
  • 遍历除首行、首列外数组元素，为0则将首行、首列对应行列的数值置0。
  • 根据首行、首列中0的位置，将对应行列全部置0。
  • 根据rowrowrow和columncolumncolumn判断首行、首列是否置0。

代码

  • 时间复杂度：O(MN)O(MN)O(MN)
  • 空间复杂度：O(1)O(1)O(1)
