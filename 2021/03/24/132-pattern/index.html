「 132 Pattern 」

单调栈 Medium 每日一题

  • 题目：给定一个整数序列：a1,a2,...,ana_1, a_2, ..., a_na1​,a2​,...,an​，一个132模式的子序列ai,aj,aka_i, a_j, a_kai​,aj​,ak​被定义为：当i<j<ki < j < ki<j<k时，ai<ak<aja_i<a_k<a_jai​<ak​<aj​。设计一个算法，当给定有nnn个数字的序列时，验证这个序列中是否含有132模式的子序列。
  • 示例：

输入：[1, 2, 3, 4]
输出：false

输入：[-1, 3, 2, 0]
输出：true
解释：序列中有 3 个132模式的的子序列: [-1, 3, 2], [-1, 3, 0] 和 [-1, 2, 0].

基本思路

由于从左向右遍历很难同时兼顾1、3、2元素位置和大小关系，因此我们可以考虑从右向左遍历，3、2的关系在逆序遍历的过程中是单调递增的，因此我们是不是只需要记录一下2的数值，在向前遍历时只要找到一个小于2的元素我们就找到了子序列。

我们逆序维护一个单调栈暂存所有可以作为2的候选元素(也是可以作为3的元素)，定义 midmidmid 记录作为2的元素，在遍历元素 nums[i]nums[i]nums[i] 时考虑：

  • 考虑 nums[i]nums[i]nums[i] 可否作为1，若 nums[i]<midnums[i]<midnums[i]<mid ，我们就找到子序列。

  • 考虑 nums[i]nums[i]nums[i] 可否作为3，将 nums[i]nums[i]nums[i] 与栈顶元素比较，若 nums[i]>temp.top()nums[i]>temp.top()nums[i]>temp.top() 则说明 temp.top()temp.top()temp.top() 可以作为2，弹出并更新 midmidmid，使得 midmidmid 为仅次于 nums[i]nums[i]nums[i] 的最大值，因为 midmidmid 越大我们找到1的可能性越大。

  • 将 nums[i]nums[i]nums[i] 入栈作为3的候选元素(也可能为新子序列2的候选元素)。
