「M」120.三角形最小路径和

发表于2021-04-21更新于2021-04-24字数统计1.3k阅读时长12分

「 Triangle 」

动态规划 Medium 美区每日一题

  • 题目：给定一个三角形 triangletriangletriangle，找出自顶向下的最小路径和。每一步只能移动到下一行中相邻的结点上。相邻的结点在这里指的是下标与上一层结点下标相同或者等于上一层结点下标 + 1的两个结点。也就是说，如果正位于当前行的下标 iii ，那么下一步可以移动到下一行的下标 iii 或 i+1i + 1i+1 。
  • 示例：

输入：triangletriangletriangle = [[2], [3, 4], [6, 5, 7], [4, 1, 8, 3]]
输出：11
解释：The triangle looks like:
2
3 4
6 5 7
4 1 8 3
The minimum path sum from top to bottom is 2 + 3 + 5 + 1 = 11

自顶向下思路

我们定义 dp[i][j]dp[i][j]dp[i][j] 存储到达元素 triangle[i][j]triangle[i][j]triangle[i][j] 的最小路径和

对于元素的不同位置，我们分情况讨论路径：

  • 考虑到达 triangle[i][0]triangle[i][0]triangle[i][0] 的路径，即每一行的第一个元素，我们只能通过上一行的第一个元素移动到达，考虑最左侧元素，状态转移方程为：

    • dp[i][0]=dp[i−1][0]+triangle[i][0]dp[i][0] = dp[i - 1][0] + triangle[i][0]dp[i][0]=dp[i−1][0]+triangle[i][0]

  • 考虑到达 triangle[i][i]triangle[i][i]triangle[i][i] 的路径，即每一行的最后一个元素(三角形矩阵第 iii 行有 i+1i+1i+1 个元素，索引为 triangle[i][i]triangle[i][i]triangle[i][i])，我们只能通过上一行的最后一个元素移动到达，考虑最右侧元素，状态转移方程为：

    • dp[i][i]=dp[i−1][i−1]+triangle[i][i]dp[i][i] = dp[i - 1][i - 1] + triangle[i][i]dp[i][i]=dp[i−1][i−1]+triangle[i][i]

  • 其他情况，对于triangle[i][j]triangle[i][j]triangle[i][j] ，可以通过上一行 triangle[i−1][j−1]triangle[i- 1][j - 1]triangle[i−1][j−1] 和 triangle[i−1][j]triangle[i - 1][j]triangle[i−1][j] 到达，我们采用其中更小元素作为路径和：

    • dp[i][j]=min(dp[i−1][j−1],dp[i−1][j])+triangle[i][j];dp[i][j] = min(dp[i - 1][j - 1], dp[i - 1][j])+ triangle[i][j];dp[i][j]=min(dp[i−1][j−1],dp[i−1][j])+triangle[i][j];

假设三角形有 nnn 行，填充完 dpdpdp 数组之后，结果是 dpdpdp 数组最后一行的最小值。

代码

  • 时间复杂度：O(n2)O(n^2)O(n2)，遍历 nnn 行二维数组耗时
  • 空间复杂度：O(n2)O(n^2)O(n2)，建立 n∗nn*nn∗n 的 dpdpdp 数组存放状态

自底向上思路

我们还可以进一步降低空间复杂度，采用自底向上思路，假设三角形有 nnn 行

我们取三角形最后一行 triangle[n−1][0:n−1]triangle[n - 1][0 : n - 1]triangle[n−1][0:n−1] 作为 dpdpdp 数组，从倒数第二行开始，对于 triangle[i][j]triangle[i][j]triangle[i][j] ，可以到达下一行的 triangle[i+1][j]triangle[i + 1][j]triangle[i+1][j] 和 triangle[i+1][j+1]triangle[i + 1][j + 1]triangle[i+1][j+1] ，而此时 dp[j]dp[j]dp[j] 存储的是 triangle[i+1][j]triangle[i + 1][j]triangle[i+1][j] 的数值； dp[j+1]dp[j + 1]dp[j+1] 存储的是 triangle[i+1][j+1]triangle[i + 1][j + 1]triangle[i+1][j+1] 的数值，我们取更小值作为路径和，当我们进行完这一步操作以后，dpdpdp 中存储的第 i+1i+1i+1 行的第 jjj 个元素就失去了作用，因为当前行也就是第 iii 行没有元素可以到达 i+1i+1i+1 行的第 jjj 个元素，所以覆盖 dp[j]dp[j]dp[j]，状态转移方程为：

  • dp[j]=min(dp[j],dp[j+1])+triangle[i][j];dp[j] = min(dp[j], dp[j + 1]) + triangle[i][j];dp[j]=min(dp[j],dp[j+1])+triangle[i][j];
