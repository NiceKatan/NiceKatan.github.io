「 Increasing Order Search Tree 」

树 DFS Easy 每日一题

  • 题目：给你一棵二叉搜索树，请你按中序遍历将其重新排列为一棵递增顺序搜索树，使树中最左边的节点成为树的根节点，并且每个节点没有左子节点，只有一个右子节点。
  • 示例：

输入：rootrootroot = [5, 3, 6, 2, 4, null, 8, 1, null, null, null, 7, 9]
输出：[1, null, 2, null, 3, null, 4, null, 5, null, 6, null, 7, null, 8, null, 9]

方法一：额外空间

辅助栈模拟中序遍历

迭代中序遍历解法可参考

代码

  • 时间复杂度：O(n)O(n)O(n)
  • 空间复杂度：O(n)O(n)O(n)，需要辅助栈，平均时间复杂度为O(logn)O(logn)O(logn)，最坏情况下为O(n)O(n)O(n)

递归中序遍历数组保存节点值，根据数组中节点值创建树

  • 时间复杂度：O(n)O(n)O(n)，nnn 为二叉搜索树节点总数
  • 空间复杂度：O(n)O(n)O(n)，需要长度为 nnn 的数组存储节点数值
