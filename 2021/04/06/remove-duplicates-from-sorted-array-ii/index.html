「 Remove Duplicates From Sorted Array II 」

双指针 Medium 每日一题

  • 题目：给你一个有序数组 numsnumsnums，请你原地删除(使用 O(1) 额外空间)重复出现的元素，使每个元素最多出现两次，返回删除后数组的新长度。

numsnumsnums 是用"引用"方式传递的，这意味着在函数里修改输入数组对于调用者是可见的。根据函数返回的长度，会打印出数组中该长度范围内的所有元素

  • 示例：

输入：numsnumsnums = [1,1,1,2,2,3]
输出：5, numsnumsnums = [1,1,2,2,3]
解释：函数应返回新长度 lengthlengthlength = 5, 并且原数组的前五个元素被修改为[1,1,2,2,3] 。 不需要考虑数组中超出新长度后面的元素。

基本思路

本题要求使用O(1)O(1)O(1)的额外空间完成修改元素的任务，意味着我们只能修改原始数组，重复出现2次以上的元素需要被后面元素覆盖，我们在遍历的时候就需要一个指针指向新数组放置元素位置，一个指针指向旧数组遍历的位置，因此本题采用双指针解法。

定义指针 slowslowslow 指向新数组将要追加元素位置，指针 fastfastfast 指向旧数组当前遍历位置，本题允许重复两次，所以默认保留前两位，slowslowslow 与 fastfastfast 初值均为2，即 slowslowslow：即将追加第三个元素，fastfastfast：正遍历第三个元素。

对于每一次判断，我们比较 nums[slow−2]nums[slow-2]nums[slow−2] 与 nums[fast]nums[fast]nums[fast]，判断当前遍历元素与新数组写入位置前的第2个元素(上上个元素)是否相同：

  • nums[slow−2]=nums[fast]nums[slow-2]=nums[fast]nums[slow−2]=nums[fast]，说明如果覆盖数组，存在 nums[slow−2]=nums[slow−1]=nums[slow]nums[slow-2]=nums[slow-1]=nums[slow]nums[slow−2]=nums[slow−1]=nums[slow] 三位相同数字，不符合题意，所以不可写入元素，因此 slowslowslow 保持不动，fastfastfast 移动寻找下一个符合元素。
  • 当且仅当 nums[slow−2]≠nums[fast]nums[slow-2]\neq nums[fast]nums[slow−2]​=nums[fast] 时，nums[slow]=nums[fast]nums[slow]=nums[fast]nums[slow]=nums[fast] 覆盖元素，slowslowslow++、fastfastfast++快慢指针均移动。

本题还可拓展为「保留k位」，修改快慢指针差值或撰写函数可实现，参考大佬@宫水三叶。
